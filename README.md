# Smart Trading Assistant (Expo + React Native)

Mobile trading companion focused on learning and practice:

- Live market watchlist (Binance API with fallback data).
- Rule-based signal engine (trend + RSI + momentum score).
- Risk-based paper trading with position sizing.
- Portfolio tracking with open/closed PnL.
- Built-in trading lessons + quizzes with saved progress.

## Tech Stack

- Expo + React Native + TypeScript
- React Navigation (bottom tabs)
- Zustand + AsyncStorage persistence

## Run

1. Install dependencies:
   `npm install`
2. Start Expo:
   `npm run start`
3. Optional:
   `npm run android`
   `npm run ios`
   `npm run web`

## Notes

- This app is educational and for practice workflow.
- No AI chat feature is included.
